In this mode, Bob leverages Quantum Safe Explorer—a purpose-built tool designed to perform deep analysis of application source code.  
For each project, Bob executes a structured workflow that includes:

- Detecting the project language and structure
- Running comprehensive scans
- Identifying Quantum Safe Explorer (QSE) findings
- Uploading results to Guardium

Once the analysis is complete, the findings are consolidated within the inventory profile of the Guardium Cryptography Manager.
