Get Started with IBM Bob – Everything You Need to Know

IBM Bob is an AI-powered development assistant built on IBM technology. This section covers the fundamentals — what Bob is, how it works, and how to get the most out of it as an IBM Business Partner.

What is IBM Bob?

Bob is a customizable AI coding and development assistant tailored for IBM technologies. It combines large language model capabilities with IBM-specific knowledge, tooling, and best practices to accelerate solution delivery across the IBM Data & AI and Automation portfolios.

Core Concepts

🤖 Modes

Modes define Bob's persona, focus, and behaviour for a specific domain or task. Switch between modes to get contextually relevant assistance — from Carbon UI development to watsonx AI workflows.

🛠️ Skills

Skills are reusable, composable capabilities that extend what Bob can do. Attach skills to any mode to unlock specialised actions such as generating test plans, scaffolding microservices, or producing architecture diagrams.

🤝 SubAgents

SubAgents are autonomous task runners that Bob can orchestrate to handle long-running or parallel workloads — from running end-to-end tests to deploying to IBM Cloud environments.
