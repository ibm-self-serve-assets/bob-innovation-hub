🛠️ Skills

What are Bob Skills?

Skills are reusable, composable capabilities that extend Bob beyond its default behaviour. A skill can be attached to any mode and teaches Bob a focused, specialist task — such as generating architecture diagrams, scaffolding a microservice, producing SBOM reports, or writing test plans.

Skills are defined in SKILL.md files and activated on demand. They follow a consistent frontmatter schema so they can be discovered, versioned, and shared across teams.

🔧 How to Use a Skill

  1. Open a Bob session with your chosen Mode active
  2. Type use skill: <skill-name> or activate via the sidebar
  3. Bob loads the skill's instructions into context
  4. Proceed with your task — Bob now has the skill's specialised knowledge

✏️ How to Create a Skill

  1. Create a SKILL.md file in your project's .bob/skills/ directory
  2. Add the required frontmatter: name, description, version
  3. Write the skill body — clear, imperative instructions Bob will follow
  4. Commit and share with your team or submit to this hub
