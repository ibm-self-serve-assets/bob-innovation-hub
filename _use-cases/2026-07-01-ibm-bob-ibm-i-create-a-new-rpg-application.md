In this demo, we watch Bob generate a complete student management application — including DB2 database design, an RPG backend, a Python Flask web services layer, a responsive HTML5 frontend, testing, deployment scripts, and full documentation.  

Bob works through each phase methodically, starting with a high-level design document that covers database architecture, backend structure, web services, and frontend layout. From there, Bob builds out every layer of the application one component at a time.
