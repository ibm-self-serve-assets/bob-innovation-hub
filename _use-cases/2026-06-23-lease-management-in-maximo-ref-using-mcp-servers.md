**MREF Contract Manager** is a comprehensive Model Context Protocol (MCP) server that enables AI assistants like IBM Bob to manage real estate contracts and leases in IBM Maximo Real Estate and Facilities (TRIRIGA). It provides 10 powerful tools for contract management including creating, updating, filtering, exporting, bulk importing, analyzing statistics, searching, and generating reports—all through OSLC REST APIs. Built with Python 3.10+ and FastMCP framework, it offers production-ready integration with authentication, session management, and comprehensive error handling for seamless real estate contract lifecycle management.
