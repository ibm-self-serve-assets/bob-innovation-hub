This demo showcase an end-to-end API lifecycle automation workflow using AI and IBM API management capabilities. It covered generating an enterprise-grade OpenAPI specification, validating it against OWASP security standards, automatically generating and deploying the backend implementation on IBM Cloud Code Engine, and updating the API specification to integrate with the deployed backend services. It also highlights the integration of API Connect MCP Server with Bob to streamline API management activities such as listing projects, creating and deploying APIs, and testing APIs directly within API Connect. Overall, the solution demonstrates how AI-driven automation can accelerate secure API development, deployment, governance, and testing in an enterprise environment.
