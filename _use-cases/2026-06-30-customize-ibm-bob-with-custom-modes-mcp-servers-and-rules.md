Learn how to tailor Bob's behaviour for specific use cases. Custom modes restrict which tools Bob can access. MCP servers extend Bob's knowledge with live, up-to-date data — like the Quarkus MCP server for correct dependency versions. Rules files let you teach Bob your team's exact coding conventions, so it generates the code you want, not just what it was trained on.
